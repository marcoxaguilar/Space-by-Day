# Space by Day Design — SEO-Optimized Website

Static website for **Space by Day Design**, an interior design studio based in Lemont, IL serving the Chicago metro area.

## Deployment

- **Platform:** Netlify (via GitHub integration)
- **Publish directory:** `.` (root)
- **Build command:** none — static HTML
- **Custom domain:** `www.spacebyday.com`

## Files

| File | Purpose |
|------|---------|
| `index.html` | Full website — homepage + 5 service pages + portfolio + builders + about |
| `blog-kitchen-remodel-cost-chicago.html` | Blog: Kitchen remodel cost guide Chicago 2026 |
| `blog-what-are-3d-interior-renderings.html` | Blog: What are 3D interior renderings |
| `sitemap.xml` | XML sitemap (12 URLs) for Google/Bing |
| `robots.txt` | Crawler directives + sitemap reference |
| `netlify.toml` | Netlify config, security headers, caching |
| `_redirects` | Clean URLs + blog routes + old Wix 301s |
| `_headers` | Security headers |
| `404.html` | Custom error page |
| `manifest.json` | PWA metadata |
| `README.md` | This file |
| `.gitignore` | Git ignore rules |
